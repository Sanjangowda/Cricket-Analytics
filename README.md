# Cricket-Analytics
## Analyzing 2022 T20WC data and selecting the final team through visualization
---

This was my first sports data analysis, It was very exciting and complex at the same time, but the final result was worth it.

- Data Scraping from ESPN cricinfo website was the big challenge.
- It was in JSON format, So I converted it into CSV format and cleaned it using Python. 
- Next I transformed the data, Modeled it and created the required Key Measures using DAX formulas.
- Created Custom Columns to support the measures.
- Built actionable multiple dashboard pages.  
- Interlinked all dashboards for interactive navigation between them.

  ---
  ## Follow the below steps to try this project:
  > STEP 1:
  - Clone the repository or download it.
  - Web Scrape the data from ESPN Cricinfo website using 'web_scrapping_code' file in repository.
  > STEP 2:
  - Use the 'cricket_analytics.ipynb' jupyter notebook from the repository and install all the dependencies.
  - Run every cells in the notebook one by one
  > STEP 3:
  - Convert the data from JSON format to CSV format and download the cleaned and transformed data.
  - Or else just download the ready-made CSV file from the repository.
  - Open Power BI and connect the data
  > STEP 4 :
  - Transform the data according to your requirements.
  - Create Key Measures and Custom colomns.
  > STEP 5:
  - Build the actionable dashboards of your own design choice.
  - Interlink all the dashboard pages for the navigation between them.
